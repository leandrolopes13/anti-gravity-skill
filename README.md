# LLM Cost Efficiency Skill

[![Compatible with Claude Code](https://img.shields.io/badge/Compatible%20with-Claude%20Code-vibrantgreen.svg)](https://claude.ai)
[![AgentSkillsManager](https://img.shields.io/badge/AgentSkills-Manager-blue.svg)](https://marketplace.visualstudio.com/items?itemName=whyuds.agentskillsmanager)

**LLM Cost Efficiency** (Economia de LLM) é uma skill projetada para ajudar desenvolvedores e agentes a reduzir drasticamente o consumo de tokens e otimizar custos operacionais. Esta skill fornece diretrizes práticas, estratégias de engenharia de prompt e ferramentas de estimativa.

## 🚀 Estrutura da Skill

A skill está organizada em módulos temáticos localizados na pasta `skills/LLMCostEfficiencySkill/`. O ponto de entrada principal é o arquivo `SKILL.md`.

### Módulos Disponíveis:

- 💡 **[Estratégias de Economia](skills/LLMCostEfficiencySkill/strategies.md)**: O Princípio da Necessidade e cache mental.
- ✍️ **[Prompt Engineering](skills/LLMCostEfficiencySkill/prompt_engineering.md)**: Técnicas de concisão e formatos leves.
- 📦 **[Context Management](skills/LLMCostEfficiencySkill/context_management.md)**: Chunking, carregamento parcial e resets de contexto.
- ⚡ **[Model Features](skills/LLMCostEfficiencySkill/model_features.md)**: Context Caching (Gemini/Anthropic) e Batch APIs.
- 📋 **[Preflight Checklist](skills/LLMCostEfficiencySkill/preflight_checklist.md)**: Checklist para validação de eficiência antes do envio.

## 📦 Instalação (VSCode)

Esta skill é totalmente compatível com a extensão **[AgentSkillsManager](https://marketplace.visualstudio.com/items?itemName=whyuds.agentskillsmanager)**.

1. Instale a extensão `AgentSkillsManager` no seu VSCode.
2. Nas configurações da extensão (`settings.json`), aponte para o diretório local ou repositório desta skill.
3. Na aba da extensão, instale a **LLMCostEfficiencySkill**.

## 💡 Como Usar

Uma vez instalada, a skill será carregada automaticamente pelo agente. Consulte os módulos sempre que estiver lidando com grandes volumes de dados ou quando precisar otimizar o custo de uma automação complexa.
