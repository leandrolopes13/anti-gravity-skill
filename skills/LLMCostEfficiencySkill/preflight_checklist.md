# Checklist de Pré-Voo (Eficiência)

Antes de enviar um prompt complexo ou volumoso, faça estas 5 perguntas:

1. [ ] **Eu realmente preciso de todos estes arquivos?** (Remova arquivos que não serão lidos ou modificados).
2. [ ] **O contexto está limpo?** (Removi logs antigos, conversas irrelevantes ou stack traces redundantes?).
3. [ ] **Instruí o modelo a ser conciso?** (Adicionei "Apenas o código" ou "Responda sem explicações" se apropriado?).
4. [ ] **Estou usando o modelo certo?** (Esta tarefa poderia ser resolvida por um modelo menor e mais barato?).
5. [ ] **Posso usar ferramentas locais primeiro?** (Já usei `grep`, `lint` ou `tests` para filtrar a informação?).
